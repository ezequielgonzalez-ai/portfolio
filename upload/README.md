# 🚀 Matias Ezequiel Gonzalez - Portfolio

Portfolio profesional con estética **Dark Bento Modernism**, optimizado para conversión y rendimiento.

![Next.js](https://img.shields.io/badge/Next.js-16-black?style=flat-square&logo=next.js)
![TypeScript](https://img.shields.io/badge/TypeScript-5-blue?style=flat-square&logo=typescript)
![Tailwind CSS](https://img.shields.io/badge/Tailwind-4-38B2AC?style=flat-square&logo=tailwind-css)
![Framer Motion](https://img.shields.io/badge/Framer_Motion-11-FF0050?style=flat-square&logo=framer)

## ✨ Características

- 🌓 **Dark/Light Mode** - Persistencia en localStorage
- ✨ **Text Generate Effect** - Animación de revelación de texto
- 🧲 **Magnetic Buttons** - Botones con efecto magnético
- 📜 **Infinite Scroll** - Tech stack con scroll infinito
- 💎 **Glassmorphism** - Tarjetas con efecto cristal
- 🌌 **Aurora Background** - Fondo animado estilo aurora
- 📱 **Responsive** - Diseño adaptable a todos los dispositivos
- 🎯 **SEO Optimizado** - Sitemap dinámico, robots.txt, meta tags

## 🛠️ Stack Tecnológico

- **Framework:** Next.js 16 (App Router)
- **Lenguaje:** TypeScript
- **Estilos:** Tailwind CSS 4
- **Animaciones:** Framer Motion
- **Componentes:** shadcn/ui
- **Iconos:** Lucide React

## 🚀 Inicio Rápido

```bash
# Clonar el repositorio
git clone https://github.com/TU_USUARIO/portfolio.git

# Entrar al directorio
cd portfolio

# Instalar dependencias
bun install

# Iniciar servidor de desarrollo
bun run dev

# Build para producción
bun run build
```

## 📁 Estructura del Proyecto

```
src/
├── app/
│   ├── page.tsx          # Página principal
│   ├── layout.tsx        # Layout con SEO
│   ├── globals.css       # Estilos globales
│   ├── sitemap.ts        # Sitemap dinámico
│   └── robots.ts         # Robots.txt
├── components/
│   └── portfolio/
│       ├── hero.tsx      # Sección hero
│       ├── nav.tsx       # Navegación
│       ├── bento-grid.tsx # Grid de servicios
│       ├── tech-stack.tsx # Stack tecnológico
│       ├── social-proof.tsx # Testimonios
│       ├── experience.tsx # Experiencia
│       ├── footer.tsx    # Pie de página
│       └── ...
└── lib/
    ├── portfolio-data.ts # Configuración centralizada
    └── utils.ts          # Utilidades
```
